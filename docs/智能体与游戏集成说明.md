# 智能体与游戏集成说明

## 功能概述

智能体与游戏已完全打通，形成完整的学习闭环：

1. **智能体 → 游戏**：智能体提供游戏链接
2. **游戏 → 智能体**：游戏完成后，用户返回智能体，智能体给出评价和建议

---

## 使用流程

### 步骤1：启动智能体

对智能体说："开始游戏" 或 "我想玩游戏"

### 步骤2：选择难度

智能体提供以下选项：
- **简单**：两个数加减法（两位数）
- **中等**：三个数加减法（两位数）
- **困难**：三个数加减法（三位数）

### 步骤3：进入游戏

点击智能体提供的链接，打开游戏页面

游戏页面会显示提示：
```
🤖 这是来自智能体的简便运算游戏
完成后点击"返回智能体"按钮，我会根据你的表现给出评价！
```

### 步骤4：玩游戏

- 每关10道题
- 答对+10分，答错-5分
- 选择方法、列式子、给结果
- 错题有闪电老师详细讲解

### 步骤5：完成游戏

游戏结束后，点击"返回智能体"按钮

系统会显示提示：
```
🎮 游戏结果已保存！

请返回智能体，告诉我"游戏已完成"，我会根据你的表现给出评价和建议！
```

### 步骤6：返回智能体

对智能体说："游戏已完成"

智能体会询问：
```
太好了！请告诉我你的游戏结果：
- 得分是多少分？
- 做对了多少题？
- 选择的难度是什么？
```

### 步骤7：智能体评价

根据用户提供的游戏结果，智能体会给出：
- 🎯 综合评价（完美通关/优秀/良好/及格/继续努力）
- ⭐ 鼓励话语
- 💡 学习建议
- 📈 下一步练习方向

## 智能体工具

### 1. start_game_session

**功能：** 开始游戏会话，提供游戏链接

**参数：**
- `difficulty`: 游戏难度（easy/medium/hard）

**返回：**
```json
{
  "success": true,
  "session_id": "game_xxx",
  "game_link": "https://...",
  "difficulty": "easy",
  "message": "游戏已开始！请打开链接玩游戏..."
}
```

### 2. game_completed

**功能：** 用户完成游戏后报告结果，智能体给出评价

**参数：**
- `score`: 游戏得分
- `correct`: 正确题目数
- `total`: 总题目数
- `difficulty`: 游戏难度（easy/medium/hard）

**返回：**
```json
{
  "success": true,
  "evaluation": "优秀",
  "emoji": "🥇",
  "comment": "做得非常出色！你掌握得很好！",
  "encouragement": "再多练习几次，你肯定能拿到满分！",
  "score": 85,
  "correct": 9,
  "total": 10,
  "accuracy": 90,
  "difficulty": "中等（三个数）",
  "tips": [
    "💡 建议：试试用不同的方法解题，找到最适合你的方法。",
    "💡 建议：练习三个数加减法时，先算哪两个数更方便？"
  ]
}
```

### 3. get_game_tips

**功能：** 获取简便运算技巧和学习建议

**参数：**
- `topic`: 主题（all/basic/advanced）

**返回：**
```json
{
  "success": true,
  "topic": "all",
  "basic_tips": [...],
  "advanced_tips": [...]
}
```
