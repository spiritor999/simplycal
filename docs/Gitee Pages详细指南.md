# 🚀 Gitee Pages 详细启用指南

## 📋 正确的启用步骤

### 方法1：通过仓库页面启用（推荐）

#### 第1步：打开仓库
访问：https://gitee.com/spiritor/simplecal

#### 第2步：找到 Gitee Pages 入口

**位置可能在以下地方之一：**

1. **仓库顶部导航栏**
   - 查找标签：服务 / Pages / Gitee Pages
   - 点击进入

2. **仓库右侧边栏**
   - 查找 "Gitee Pages" 链接
   - 点击进入

3. **仓库设置**
   - 点击右上角的 "设置" ⚙️
   - 在左侧菜单中查找 "Gitee Pages"
   - 点击进入

#### 第3步：启动服务

进入 Gitee Pages 页面后：

1. **点击 "启动" 按钮**
   - 如果是首次使用，会提示开通服务

2. **选择部署配置**
   - 部署分支：选择 `main`
   - 部署目录：选择 `/`（根目录）

3. **点击 "启动"**
   - 等待 1-2 分钟
   - 查看部署状态

#### 第4步：获取访问地址

部署成功后，会显示访问地址：
```
https://spiritor.gitee.io/simplecal/
```

游戏链接：
```
https://spiritor.gitee.io/simplecal/game.html
```

---

### 方法2：通过 Gitee 个人中心

#### 第1步：访问个人中心
打开：https://gitee.com/

#### 第2步：进入服务管理

1. 点击右上角头像
2. 选择 "设置" 或 "个人中心"
3. 查找 "Gitee Pages" 或 "我的服务"

#### 第3步：找到项目并启用

1. 在 Gitee Pages 列表中查找项目
2. 找到 `spiritor/simplecal`
3. 点击 "启动" 或 "启用"

---

## 🔍 如果还是找不到 Gitee Pages

### 可能的原因：

1. **账户类型限制**
   - Gitee Pages 可能只对某些账户类型开放
   - 需要实名认证
   - 需要绑定手机号

2. **仓库权限**
   - 需要是公开仓库
   - 需要有管理员权限

3. **服务未开通**
   - 需要先在个人设置中开通 Gitee Pages 服务

## 🎯 备选方案

### 方案1：使用 GitHub Pages（国内可访问）

虽然速度慢，但完全可以使用！

**步骤：**

1. 访问：https://github.com/spiritor999/simplycal/settings/pages

2. 配置：
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
   - 点击 **Save**

3. 等待 1-2 分钟

4. 访问：
   ```
   https://spiritor999.github.io/simplycal/game.html
   ```

---

### 方案2：使用 Coding Pages（国内服务）

Coding 是国内 Git 平台，Pages 速度快！

**步骤：**

1. 注册 Coding 账号：https://coding.net

2. 创建项目或导入 Git 仓库

3. 启用 Pages 服务

4. 获取访问地址

---

### 方案3：使用 Netlify（国外但速度快）

虽然服务器在国外，但 CDN 加速效果好！

**步骤：**

1. 访问：https://app.netlify.com/drop

2. 拖拽项目文件夹到网页

3. 等待部署

4. 获得公网链接

---

### 方案4：使用 Coze 对象存储（推荐！）

既然游戏已经在 Coze 环境中，可以直接上传到对象存储！

**需要使用 Coze 的存储能力：**

1. 使用技能中的 storage 集成
2. 上传游戏文件到对象存储
3. 获得公网访问链接

让我帮你实现这个方案！
