# 🔧 方法选项优化

## ✅ 已完成的优化

根据题目类型，动态显示相关的简便运算方法，避免出现不相关的方法选项。

---

## 📋 问题描述

**之前的做法：**
无论题目是加减法还是乘除法，下拉菜单都显示所有方法：
```
-- 请选择方法 --
凑十法
破十法
加法交换律
加法结合律
乘法分配律
乘法结合律
乘法交换律
除法分配律
直接计算
```

**存在的问题：**
- ❌ 加法题目显示乘法方法，容易混淆
- ❌ 减法题目显示乘除法方法，不符合逻辑
- ❌ 学生可能选择错误的方法
- ❌ 影响教学效果

## 🔧 技术实现

### 1. 题目生成时保存运算符

```javascript
return {
    question: question,
    answer: answer,
    method: method.name,
    operator: method.operator,  // 保存运算符类型：+ - × ÷
    explanation: this.getExplanation(question, answer, method.name)
};
```

### 2. 定义方法映射

```javascript
// 根据运算符获取可选方法
getAvailableMethods: function(operator) {
    const allMethods = {
        '+': ['凑十法', '加法交换律', '加法结合律', '直接计算'],
        '-': ['破十法', '加法交换律', '加法结合律', '直接计算'],
        '×': ['乘法分配律', '乘法结合律', '乘法交换律', '直接计算'],
        '÷': ['乘法分配律', '除法分配律', '乘法结合律', '直接计算']
    };
    return allMethods[operator] || ['直接计算'];
},
```

### 3. 显示题目时动态生成选项

```javascript
// 动态更新方法选项
function updateMethodOptions(operator) {
    const methodSelect = document.getElementById('method');
    const availableMethods = problemGenerator.getAvailableMethods(operator);
    
    // 清空现有选项
    methodSelect.innerHTML = '<option value="">-- 请选择方法 --</option>';
    
    // 添加可用方法选项
    availableMethods.forEach(method => {
        const option = document.createElement('option');
        option.value = method;
        option.textContent = method;
        methodSelect.appendChild(option);
    });
}
```

### 4. 在 showQuestion 中调用

```javascript
function showQuestion() {
    const question = gameState.questions[gameState.currentQuestionIndex];
    
    // ...
    
    // 根据运算符动态生成方法选项
    updateMethodOptions(question.operator);
    
    // ...
}
```

## 📊 方法分类

### 加减法方法
适用于 `+` 和 `-` 运算：

| 方法 | 适用运算 | 说明 |
|------|---------|------|
| 凑十法 | + | 凑成10再计算 |
| 破十法 | - | 从10中减去 |
| 加法交换律 | + | 交换加数位置 |
| 加法结合律 | +, - | 分组计算 |

### 乘除法方法
适用于 `×` 和 `÷` 运算：

| 方法 | 适用运算 | 说明 |
|------|---------|------|
| 乘法分配律 | ×, ÷ | 分配到各个项 |
| 乘法结合律 | ×, ÷ | 先计算容易的部分 |
| 乘法交换律 | × | 交换因数位置 |
| 除法分配律 | ÷ | 分配到各个项 |

### 通用方法
适用于所有运算：

| 方法 | 适用运算 | 说明 |
|------|---------|------|
| 直接计算 | +, -, ×, ÷ | 按顺序计算 |
