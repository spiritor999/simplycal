# 快速启动指南（3分钟上手）

## 🚀 最快启动方式（推荐）

### 步骤1：一键启动所有服务

```bash
bash scripts/start_all.sh
```

等待几秒钟，你会看到：

```
🎉 所有服务已启动成功！

📋 服务信息:
   智能体服务:
   - 本地访问: http://localhost:8000
   - API 文档: http://localhost:8000/docs

   游戏服务器:
   - 本地访问: http://localhost:8080/math_game.html

   智能体与游戏已打通:
   - 对智能体说'开始游戏'即可获取游戏链接
   - 游戏完成后返回智能体获取评价
```

### 步骤2：访问服务

#### 访问智能体 API 文档
打开浏览器访问：http://localhost:8000/docs

#### 访问游戏页面
打开浏览器访问：http://localhost:8080/math_game.html

### 步骤3：测试智能体

#### 方法1：使用 API 文档测试
1. 打开 http://localhost:8000/docs
2. 找到 `/run` 接口
3. 点击 "Try it out"
4. 输入 JSON: `{"query": "你好"}`
5. 点击 "Execute"

#### 方法2：使用 curl 命令
```bash
# 健康检查
curl http://localhost:8000/health

# 对话
curl -X POST http://localhost:8000/run \
  -H "Content-Type: application/json" \
  -d '{"query": "开始游戏"}'
```

### 步骤4：停止服务

```bash
bash scripts/stop_all.sh
```

## 🌐 外网访问（可选）

### 使用 ngrok 快速实现外网访问

1. **安装 ngrok**（如果还没安装）
   - macOS: `brew install ngrok`
   - Linux: 参考 `docs/智能体部署指南.md`
   - Windows: 下载 https://ngrok.com/download

2. **启动 ngrok**
   ```bash
   bash scripts/start_ngrok.sh
   ```

3. **获取公网地址**
   ngrok 会显示一个公网地址，例如：
   ```
   Forwarding  https://xxxx-xx-xx-xx-xx.ngrok-free.app -> http://localhost:8000
   ```

4. **使用公网地址访问**
   - 智能体: https://xxxx-xx-xx-xx-xx.ngrok-free.app
   - API 文档: https://xxxx-xx-xx-xx-xx.ngrok-free.app/docs

## 🔧 常用命令

### 启动服务
```bash
# 启动所有服务（智能体 + 游戏）
bash scripts/start_all.sh

# 只启动智能体服务
bash scripts/start_agent_server.sh

# 只启动游戏服务器
bash scripts/start_game_server.sh
```

### 停止服务
```bash
# 停止所有服务
bash scripts/stop_all.sh

# 手动停止（查看 PID 后）
ps aux | grep uvicorn
kill <PID>
```

### 查看日志
```bash
# 智能体日志
tail -f /tmp/agent.log

# 游戏日志
tail -f /tmp/game.log

# 应用日志
tail -f /app/work/logs/bypass/app.log
```

### 检查服务状态
```bash
# 检查端口占用
lsof -i :8000
lsof -i :8080

# 健康检查
curl http://localhost:8000/health
```

## 🎮 智能体与游戏交互流程

### 1. 启动游戏
```
对智能体说："开始游戏"

智能体返回：
{
  "success": true,
  "game_link": "https://...",
  "message": "游戏已开始！请打开链接玩游戏..."
}
```

### 2. 玩游戏
- 点击智能体提供的游戏链接
- 选择难度：简单/中等/困难
- 玩10道题
- 查看得分和讲解

### 3. 返回智能体
- 游戏完成后，点击"返回智能体"按钮
- 系统提示："游戏结果已保存！请返回智能体..."

```
对智能体说："游戏已完成，得了85分，做对9题，难度中等"

智能体返回：
{
  "success": true,
  "evaluation": "优秀",
  "emoji": "🥇",
  "comment": "做得非常出色！你掌握得很好！",
  "tips": [
    "💡 建议：试试用不同的方法解题..."
  ]
}
```

## ⚠️ 常见问题

### 问题1: 端口被占用
```bash
# 查看哪个进程占用了端口
lsof -i :8000

# 杀死进程
kill -9 <PID>

# 或者使用其他端口
PORT=8001 bash scripts/start_agent_server.sh
```

### 问题2: 服务启动失败
```bash
# 查看详细日志
tail -f /tmp/agent.log
tail -f /tmp/game.log

# 重新安装依赖
pip install -r requirements.txt
```

### 问题3: 无法访问服务
```bash
# 检查服务是否正在运行
curl http://localhost:8000/health

# 检查防火墙
sudo ufw allow 8000
sudo ufw allow 8080
```

### 问题4: ngrok 无法启动
```bash
# 确保 ngrok 已安装
ngrok version

# 确保 ngrok 已登录
ngrok config add-authtoken YOUR_AUTH_TOKEN

# 确保智能体服务正在运行
bash scripts/start_all.sh
```
