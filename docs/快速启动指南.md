## 🔧 启动服务器

### 一键启动（推荐）

同时启动游戏服务器和智能体服务：

```bash
bash scripts/start_all.sh
```

### 分别启动

**只启动游戏服务器：**
```bash
bash scripts/start_game_server.sh
```

**只启动智能体服务：**
```bash
bash scripts/http_run.sh -p 8000
```

### 使用Python内置服务器

```bash
python -m http.server 8080
```

## 💡 常见问题

### Q1: 端口被占用怎么办？

**检查端口占用：**
```bash
lsof -i :8080
lsof -i :8000
```

**使用其他端口：**
```bash
python -m http.server 9000
# 然后访问: http://localhost:9000/assets/math_game.html
```

### Q2: 游戏链接打不开？

**解决方案：**

1. **检查服务器是否运行**
   ```bash
   curl -I http://localhost:8080/assets/math_game.html
   ```

2. **如果未运行，启动服务器**
   ```bash
   bash scripts/start_game_server.sh
   ```

3. **或者使用文本模式**
   在智能体中说"游戏链接打不开"

### Q3: 如何在手机上访问？

1. **查看电脑IP地址**
   ```bash
   # Windows
   ipconfig

   # Linux/Mac
   ifconfig
   ```

2. **在手机浏览器访问**
   ```
   http://电脑IP:8080/assets/math_game.html
   ```

3. **确保手机和电脑在同一WiFi网络**
