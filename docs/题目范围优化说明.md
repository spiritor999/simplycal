# 题目范围优化说明

## 用户需求
- **问题**：游戏题目会出现个位数，个位数的题目可以直接计算，不需要简便运算
- **需求**：所有题目都应该至少是两位数，这样学生才能真正练习简便运算

## 优化目标
- ✅ 消除个位数题目
- ✅ 确保所有数字至少是两位数
- ✅ 保持游戏难度递进
- ✅ 保证题目可解性

## 题目生成策略

### 难度分级

#### 简单模式（Easy）
- **范围**：两位数（10-99）
- **运算类型**：加、减、乘、除

**示例：**
- 加法：`45 + 37`、`28 + 56`
- 减法：`87 - 32`、`95 - 48`
- 乘法：`23 × 5`、`47 × 3`
- 除法：`96 ÷ 12`、`84 ÷ 7`

#### 中等模式（Medium）
- **范围**：两位数（10-99）
- **运算类型**：加、减、乘、除
- **特点**：数字范围比简单模式略大

**示例：**
- 加法：`78 + 45`、`56 + 89`
- 减法：`98 - 34`、`87 - 29`
- 乘法：`34 × 7`、`52 × 6`
- 除法：`144 ÷ 18`、`156 ÷ 13`

#### 困难模式（Hard）
- **范围**：三位数（100-999）
- **运算类型**：加、减、乘、除
- **特点**：需要更复杂的简便运算技巧

**示例：**
- 加法：`345 + 234`、`567 + 189`
- 减法：`789 - 345`、`923 - 567`
- 乘法：`125 × 4`、`250 × 3`
- 除法：`1000 ÷ 125`、`800 ÷ 20`

## 代码实现

### 1. 生成两位数题目
```javascript
generateTwoDigit: function() {
    const operators = ['+', '-', '×', '÷'];
    const operator = operators[Math.floor(Math.random() * operators.length)];

    let num1, num2, answer;

    switch(operator) {
        case '+':
            // 两位数加法（10-99）
            num1 = this.randomInt(10, 99);
            num2 = this.randomInt(10, 99);
            // 确保结果不超过999
            while (num1 + num2 > 999) {
                num1 = this.randomInt(10, 99);
                num2 = this.randomInt(10, 99);
            }
            answer = num1 + num2;
            break;
        case '-':
            // 两位数减法（10-99），确保被减数大于减数
            num1 = this.randomInt(10, 99);
            num2 = this.randomInt(10, 99);
            while (num1 <= num2) {
                num1 = this.randomInt(10, 99);
                num2 = this.randomInt(10, 99);
            }
            answer = num1 - num2;
            break;
        case '×':
            // 两位数乘法（10-99 × 2-20）
            num1 = this.randomInt(10, 99);
            num2 = this.randomInt(2, 20);
            // 确保结果不超过999
            while (num1 * num2 > 999) {
                num1 = this.randomInt(10, 50);
                num2 = this.randomInt(2, 10);
            }
            answer = num1 * num2;
            break;
        case '÷':
            // 两位数除法
            // 先生成除数和商，再计算被除数
            num2 = this.randomInt(2, 20);
            answer = this.randomInt(2, 50);
            // 确保被除数不超过999
            while (num2 * answer > 999) {
                num2 = this.randomInt(2, 10);
                answer = this.randomInt(2, 99);
            }
            num1 = num2 * answer;
            // 确保被除数是两位数或以上
            while (num1 < 10) {
                num2 = this.randomInt(10, 20);
                answer = this.randomInt(2, 10);
                num1 = num2 * answer;
            }
            break;
    }

    return {
        question: `${num1} ${operator} ${num2}`,
        answer: answer,
        method: this.getRandomMethod(operator),
        operator: operator,
        explanation: this.getExplanation(`${num1} ${operator} ${num2}`, answer, this.getRandomMethod(operator))
    };
}
```

## 题目特性

### 1. 加法题目
- **简单/中等**：两个两位数相加
- **困难**：两个三位数相加
- **保证**：结果不超过999
- **适用方法**：凑十法、凑整法、加法交换律、加法结合律

### 2. 减法题目
- **简单/中等**：两位数减两位数
- **困难**：三位数减三位数
- **保证**：被减数大于减数，结果为正数
- **适用方法**：破十法、凑整法、加法交换律、加法结合律

### 3. 乘法题目
- **简单/中等**：两位数乘一位数或小两位数
- **困难**：三位数乘一位数
- **保证**：结果不超过999
- **适用方法**：凑整法、乘法分配律、乘法结合律、乘法交换律

### 4. 除法题目
- **简单/中等**：两位数除一位数或小两位数
- **困难**：三位数除一位数或两位数
- **保证**：能整除，商为整数
- **适用方法**：凑整法、乘法分配律、除法分配律、乘法结合律

## 简便运算适用性

### 为什么不能是个位数？

**个位数的问题：**
- ❌ `7 + 8`：直接口算即可，不需要简便运算
- ❌ `9 × 8`：直接背九九乘法表即可
- ❌ `16 ÷ 4`：直接背乘法口诀即可

**两位数的必要性：**
- ✅ `37 + 45`：可以用凑整法 `37 + (40 + 5) = 77 + 5 = 82`
- ✅ `98 - 29`：可以用凑整法 `98 - (30 - 1) = 68 + 1 = 69`
- ✅ `25 × 4`：可以用特殊组合 `25 × 4 = 100`
- ✅ `144 ÷ 18`：可以用除法分配律 `144 ÷ 18 = (144 ÷ 12) ÷ (18 ÷ 12) = 12 ÷ 1.5 = 8`
