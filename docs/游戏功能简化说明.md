### 中级模式（Medium）
**特点：三个数加减法**

**数字范围：两位数（10-99）**

**题目示例：**
- 三个加法：`45 + 37 + 28`、`56 + 23 + 41`
- 两个加法一个减法：`78 + 45 - 23`、`89 - 34 + 27`
- 一个加法两个减法：`95 - 28 - 17`、`67 + 45 - 32`

**适用方法：**
- 凑整法（凑成整十数）
- 加法交换律
- 加法结合律
- 直接计算

**教学目标：**
- 学会处理多个数的加减运算
- 掌握凑整法的应用
- 培养灵活运用简便运算的能力

**示例讲解（凑整法）：**
```
题目：45 + 37 + 28

方法：使用凑整法
- 观察发现 37 和 28 的个位数相加是 15，可以拆分
- 37 = 35 + 2，28 = 25 + 3
- 45 + 37 + 28 = 45 + (35 + 2) + (25 + 3)
- = (45 + 35) + (25) + 2 + 3
- = 80 + 25 + 5
- = 105 + 5
- = 110

或者更简单的方法：
- 45 + 37 + 28
- = 45 + (35 + 2) + (25 + 3)
- = (45 + 35) + 25 + 5
- = 80 + 30
- = 110
```

- 加法结合律

**教学目标：**
- 掌握三位数的简便运算
- 学会凑整法在三位数中的应用
- 培养解决复杂问题的能力

**示例讲解（凑整法）：**
```
题目：345 + 234 + 189

方法：使用凑整法
- 观察发现 234 和 189 的个位数相加是 13，可以拆分
- 234 = 230 + 4，189 = 180 + 9
- 345 + 234 + 189 = 345 + (230 + 4) + (180 + 9)
- = (345 + 230 + 180) + 4 + 9
- = (345 + 410) + 13
- = 755 + 13
- = 768

或者更聪明的方法：
- 345 + 234 + 189
- = 345 + (211 + 23) + (211 - 22)
- = 345 + 211 + 211 + 1
- = (345 + 211 + 211) + 1
- = 767 + 1
- = 768
```

## 代码实现

### 1. 简单模式：两个数加减法
```javascript
generateTwoNumberAdditionSubtraction: function() {
    // 50% 加法，50% 减法
    const isAddition = Math.random() < 0.5;
    const operator = isAddition ? '+' : '-';

    let num1, num2, answer;

    if (isAddition) {
        // 两位数加法
        num1 = this.randomInt(10, 99);
        num2 = this.randomInt(10, 99);
        // 确保结果不超过999
        while (num1 + num2 > 999) {
            num1 = this.randomInt(10, 99);
            num2 = this.randomInt(10, 99);
        }
        answer = num1 + num2;
    } else {
        // 两位数减法，确保被减数大于减数
        num1 = this.randomInt(10, 99);
        num2 = this.randomInt(10, 99);
        while (num1 <= num2) {
            num1 = this.randomInt(10, 99);
            num2 = this.randomInt(10, 99);
        }
        answer = num1 - num2;
    }

    return {
        question: `${num1} ${operator} ${num2}`,
        answer: answer,
        method: this.getRandomMethod(operator),
        operator: operator,
        explanation: this.getExplanation(`${num1} ${operator} ${num2}`, answer, this.getRandomMethod(operator))
    };
}
```

### 2. 中级和高级模式：三个数加减法
```javascript
generateThreeNumberAdditionSubtraction: function(minNum, maxNum) {
    // 生成 2-3 个运算符（即 3-4 个数）
    const operatorCount = this.randomInt(2, 3);

    let numbers = [];
    let operators = [];

    // 生成第一个数
    numbers.push(this.randomInt(minNum, maxNum));

    // 生成后续的数和运算符
    for (let i = 0; i < operatorCount; i++) {
        // 随机选择加法或减法（60% 加法，40% 减法）
        const isAddition = Math.random() < 0.6;
        operators.push(isAddition ? '+' : '-');

        // 生成下一个数
        let nextNum = this.randomInt(minNum, maxNum);

        // 如果是减法，需要确保中间结果不会为负数
        if (!isAddition) {
            let currentSum = numbers[0];
            for (let j = 0; j < i; j++) {
                if (operators[j] === '+') {
                    currentSum += numbers[j + 1];
                } else {
                    currentSum -= numbers[j + 1];
                }
            }
            // 确保减法后结果仍然非负
            while (currentSum - nextNum < 0) {
                nextNum = this.randomInt(minNum, Math.min(currentSum, maxNum));
            }
        }

        numbers.push(nextNum);
    }

    // 计算答案
    let answer = numbers[0];
    for (let i = 0; i < operators.length; i++) {
        if (operators[i] === '+') {
            answer += numbers[i + 1];
        } else {
            answer -= numbers[i + 1];
        }
    }

    // 构建题目字符串
    let questionStr = numbers[0].toString();
    for (let i = 0; i < operators.length; i++) {
        questionStr += ` ${operators[i]} ${numbers[i + 1]}`;
    }

    // 确定主要运算符（用于方法选择）
    const mainOperator = operators[operators.length - 1];

    return {
        question: questionStr,
        answer: answer,
        method: this.getRandomMethod(mainOperator),
        operator: mainOperator,
        explanation: this.getExplanation(questionStr, answer, this.getRandomMethod(mainOperator))
    };
}
```

### 3. 更新方法选项
```javascript
// 根据运算符获取可选方法（只保留加减法）
getAvailableMethods: function(operator) {
    const allMethods = {
        '+': ['凑十法', '凑整法', '加法交换律', '加法结合律', '直接计算'],
        '-': ['破十法', '凑整法', '加法交换律', '加法结合律', '直接计算']
    };
    return allMethods[operator] || ['直接计算'];
}
```
