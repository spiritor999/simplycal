## 凑整法说明

### 什么是凑整法？
凑整法是通过调整数字，使其接近整十、整百数，从而简化计算的方法。

### 适用场景

#### 1. 加法凑整
**原理**：将一个数调整到最近的整十数，然后补偿差异

**示例：**
- `29 + 35` = `(30 - 1) + 35` = `30 + 35 - 1` = `65 - 1` = `64`
- `37 + 58` = `(40 - 3) + 58` = `40 + 58 - 3` = `98 - 3` = `95`
- `19 + 26 + 31` = `(20 - 1) + (25 + 1) + (30 + 1)` = `20 + 25 + 30` = `75`

#### 2. 减法凑整
**原理**：将被减数或减数调整到整十数

**示例：**
- `82 - 29` = `82 - (30 - 1)` = `82 - 30 + 1` = `52 + 1` = `53`
- `95 - 47` = `(100 - 5) - (50 - 3)` = `100 - 50 - 5 + 3` = `50 - 2` = `48`

#### 3. 乘法凑整
**原理**：利用特殊数的乘法性质（如 25×4=100, 125×8=1000）

**示例：**
- `25 × 4` = `100`
- `25 × 8` = `25 × 4 × 2` = `100 × 2` = `200`
- `125 × 8` = `1000`
- `125 × 16` = `125 × 8 × 2` = `1000 × 2` = `2000`

#### 4. 除法凑整
**原理**：将除数调整到整十数，便于估算和计算

**示例：**
- `80 ÷ 19` ≈ `80 ÷ 20` = `4`
- `240 ÷ 49` ≈ `240 ÷ 50` = `4.8` ≈ `5`

### 修改位置

### 修改内容
```javascript
getAvailableMethods: function(operator) {
    const allMethods = {
        '+': ['凑十法', '凑整法', '加法交换律', '加法结合律', '直接计算'],
        '-': ['破十法', '凑整法', '加法交换律', '加法结合律', '直接计算'],
        '×': ['凑整法', '乘法分配律', '乘法结合律', '乘法交换律', '直接计算'],
        '÷': ['凑整法', '乘法分配律', '除法分配律', '乘法结合律', '直接计算']
    };
    return allMethods[operator] || ['直接计算'];
}
```

## 教学意义

### 为什么要学凑整法？
1. **培养数感**：帮助学生理解数字之间的关系
2. **提高计算速度**：通过凑整减少计算步骤
3. **增强估算能力**：学会快速判断结果范围
4. **灵活应用**：与其他方法结合使用

### 与其他方法的区别
- **凑十法**：主要用于个位数加法（如 8 + 5 = 13）
- **凑整法**：适用于多位数，凑成整十、整百数
- **分配律**：适用于有括号或乘除法的情况
