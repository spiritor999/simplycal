## 问题描述

- 点击智能体链接时打开的是游戏页面

## 解决方案

### 1. 创建新的上传脚本

**scripts/upload_any_file.py**
- 支持自定义文件名上传
- 支持自定义对象名称
- 避免文件名混淆

**使用方法：**
```bash

python scripts/upload_any_file.py assets/agent.html agent.html

# 上传游戏页面
python scripts/upload_any_file.py assets/math_game.html math_game.html
```

## 📝 上传脚本使用说明

### 基本语法

```bash
python scripts/upload_any_file.py <本地文件路径> <存储对象名称>
```

### 参数说明

- `<本地文件路径>`：相对于项目根目录的文件路径
- `<存储对象名称>`：在对象存储中的对象名称（用于区分不同的页面）

### 示例

```bash
# 上传智能体页面
python scripts/upload_any_file.py assets/agent.html agent.html

# 上传游戏页面
python scripts/upload_any_file.py assets/math_game.html math_game.html

# 上传其他文件
python scripts/upload_any_file.py assets/test.html test.html
```
