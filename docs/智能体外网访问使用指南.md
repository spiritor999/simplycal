## 🚀 快速使用

### 方法1：直接访问智能体聊天界面

1. **打开智能体页面**
   点击上面的智能体链接，你会看到一个漂亮的聊天界面

2. **配置 API 地址**
   - 默认 API 地址：`http://localhost:8000`
   - 如果你已经启动了智能体服务（本地或服务器），点击"测试连接"按钮
   - 如果连接成功，就可以开始聊天了

3. **开始对话**
   - 在输入框中输入你的问题
   - 点击"发送"按钮或按 Enter 键
   - 智能体会回复你的问题

4. **使用快捷按钮**
   - 点击"你好"、"讲讲简便运算"、"开始游戏"等快捷按钮快速提问

### 方法2：使用游戏页面

1. **打开游戏页面**
   点击上面的游戏链接

2. **选择难度**
   - 简单：两个数加减法
   - 中等：三个数加减法
   - 困难：三位数加减法

3. **玩游戏**
   - 选择方法、列式子、给结果
   - 查看闪电老师的详细讲解

4. **返回智能体**
   - 游戏完成后，点击"返回智能体"按钮
   - 告诉智能体你的游戏结果
   - 智能体会给出评价和建议

## ⚙️ 智能体页面配置

### 默认 API 地址
智能体页面默认使用本地 API 地址：`http://localhost:8000`

### 配置远程 API 地址

如果你有远程服务器上的智能体服务，可以修改 API 地址：

1. 在智能体页面底部找到"智能体 API 地址"输入框
2. 输入你的 API 地址，例如：
   - `http://your-server-ip:8000`
   - `https://your-domain.com`
3. 点击"测试连接"按钮
4. 如果连接成功，就可以开始使用了

### 使用 ngrok 外网访问

如果你使用 ngrok 实现智能体外网访问：

1. **启动智能体服务**
   ```bash
   bash scripts/start_all.sh
   ```

2. **启动 ngrok**
   ```bash
   bash scripts/start_ngrok.sh
   ```

3. **获取 ngrok 地址**
   ngrok 会显示一个公网地址，例如：
   ```
   Forwarding  https://xxxx-xx-xx-xx-xx.ngrok-free.app -> http://localhost:8000
   ```

4. **在智能体页面配置**
   - 将 ngrok 地址（去掉端口号）输入到 API 地址框
   - 例如：`https://xxxx-xx-xx-xx-xx.ngrok-free.app`
   - 点击"测试连接"

5. **开始使用**
   配置成功后，就可以在任何地方使用智能体了！

## 🔧 故障排查

### 问题1：无法连接到智能体 API

**症状：**
点击"测试连接"按钮后显示"连接失败"

**解决方案：**

1. **检查智能体服务是否启动**
   ```bash
   # 检查服务是否运行
   curl http://localhost:8000/health

   # 如果未启动，启动服务
   bash scripts/start_all.sh
   ```

2. **检查 API 地址是否正确**
   - 确认智能体服务运行在哪个地址
   - 确保输入的 API 地址正确

3. **检查网络连接**
   - 确保可以访问智能体 API 地址
   - 检查防火墙设置

4. **使用 ngrok（外网访问）**
   - 启动 ngrok：`bash scripts/start_ngrok.sh`
   - 使用 ngrok 提供的公网地址

### 问题2：游戏链接打不开

**症状：**
点击游戏链接后无法打开

**解决方案：**

1. **检查链接是否完整**
   - 确保链接没有被截断
   - 复制完整的链接

2. **检查网络连接**
   - 确保可以访问互联网
   - 检查网络设置

3. **尝试其他浏览器**
   - Chrome、Firefox、Safari 等

### 问题3：智能体回复慢或无响应

**症状：**
发送消息后很久没有回复

**解决方案：**

1. **检查智能体服务状态**
   ```bash
   # 查看日志
   tail -f /tmp/agent.log
   ```

2. **检查网络延迟**
   - 如果使用远程 API，检查网络延迟
   - 尝试使用本地 API

3. **刷新页面**
   - 刷新智能体页面
   - 重新连接
