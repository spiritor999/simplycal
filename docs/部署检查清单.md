## 🚀 下一步操作

### 第1步：推送代码到Gitee ⏳

**在你的终端执行：**
```bash
cd /workspace/projects
git push -u origin main
```

系统会提示输入：
- 用户名：`spiritor`
- 密码：你的Gitee密码或访问令牌

**详细指南：** `docs/推送到Gitee指南.md`

---

### 第2步：验证推送成功

推送完成后，访问：https://gitee.com/spiritor/simplecal

确认能看到所有文件和提交历史。

---

### 第3步：部署智能体到Coze平台

1. 登录Coze平台
2. 创建新工作流
3. 选择"从Git导入"
4. 输入仓库地址：`https://gitee.com/spiritor/simplecal.git`
5. 选择分支：`main`
6. 配置环境变量
7. 点击部署

**详细指南：** `docs/Coze平台部署指南.md`

---

### 第4步：部署游戏页面到Vercel

```bash
# 安装Vercel CLI
npm install -g vercel

# 登录
vercel login

# 部署
cd /workspace/projects
vercel
```

记录返回的链接，例如：`https://abc123.vercel.app/game.html`

---

### 第5步：更新游戏链接

编辑 `src/tools/game_interaction_tool.py`:

```python
http_link = "https://your-project.vercel.app/game.html"
```

重新提交并推送：
```bash
git add .
git commit -m "feat: 更新游戏链接为Vercel地址"
git push
```

## 💡 重要提示

### 推送时可能遇到的问题

**问题1：身份验证失败**
- 解决：使用访问令牌而不是密码
- 参考：`docs/推送到Gitee指南.md`

**问题2：推送被拒绝**
- 解决：使用 `git push -u origin main --force`
- ⚠️ 谨慎使用，会覆盖远程内容

**问题3：网络超时**
- 解决：检查网络连接，或重试
