## 🔧 链接过期处理

### 当前链接有效期
- **有效期**：7 天
- **过期时间**：从生成时间起 7 天后

### 如何续期？

**方法1：重新上传（推荐）**

```python
from coze_coding_dev_sdk.s3 import S3SyncStorage

storage = S3SyncStorage()

# 重新上传游戏文件
with open('game.html', 'rb') as f:
    key = storage.upload_file(
        file_content=f.read(),
        file_name='math_game.html',
        content_type='text/html',
    )

# 生成新的签名 URL
new_url = storage.generate_presigned_url(key=key, expire_time=604800)
print(f'新的游戏链接: {new_url}')
```

**方法2：延长有效期**

在生成签名 URL 时，设置更长的过期时间：

```python
# 生成 30 天有效期的链接
url = storage.generate_presigned_url(key=key, expire_time=2592000)
```
