# 🎮 简便运算大冒险 - 游戏使用说明

## 访问游戏页面

### 方法1：直接打开HTML文件（最简单）
1. 在电脑上找到 `assets/math_game.html` 文件
2. 双击文件，用浏览器打开
3. 开始玩游戏！🎉

### 方法2：通过HTTP服务器访问
如果需要分享给其他人，可以启动HTTP服务器：

```bash
# 在项目根目录执行
python -m http.server 8080
```

然后在浏览器访问：
```
http://localhost:8080/assets/math_game.html
```

## 游戏功能介绍

### 🎯 游戏模式
- **简单模式**：两位数加减运算
- **中等模式**：三位数运算，需要凑整法
- **困难模式**：混合运算和乘法巧算

### 🌟 游戏特点
1. **3关闯关模式**：每关10分，总分30分
2. **星星奖励**：答对一关获得一颗星星
3. **实时反馈**：答题后立即显示对错
4. **精美动画**：流畅的界面动画效果
5. **响应式设计**：支持电脑和手机访问

### 🎮 如何玩
1. 选择难度（简单/中等/困难）
2. 点击"开始游戏"
3. 查看题目，输入答案
4. 点击"提交答案"或按回车键
5. 完成所有3关，查看最终得分！

### 🏆 评分标准
- **30分（满分）**：🏆 完美通关 - 简便运算小天才！
- **21-29分**：🥇 太棒了 - 表现非常出色！
- **15-20分**：🥈 做得不错 - 继续加油！
- **15分以下**：💪 继续努力 - 多练习几次！

## 常见问题

**Q: 如何分享给同学？**
A: 将 `math_game.html` 文件发送给同学，让他们用浏览器打开即可。

**Q: 题目可以自定义吗？**
A: 目前支持简单、中等、困难三种预设难度，题目随机生成。

**Q: 可以玩多次吗？**
A: 可以！游戏结束后点击"再玩一次"按钮即可重新开始。

**Q: 手机上能玩吗？**
A: 可以！游戏支持手机浏览器访问。

## 开发说明

游戏使用纯前端技术实现：
- HTML5：页面结构
- CSS3：样式和动画
- JavaScript：游戏逻辑

题目生成逻辑内置在JavaScript中，无需后端API支持，可以独立运行。
